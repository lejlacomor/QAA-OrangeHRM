# OrangeHRM Automation Project

##  Project Overview

This project is an end-to-end automation testing framework for the **OrangeHRM** web application, built using **Playwright** and **TypeScript**.

Its purpose is to automate core HRM functionalities to ensure application stability, functional accuracy, and efficient regression testing through reliable automated test execution.

The framework follows best practices such as the **Page Object Model (POM)**, clean code structure, and CI integration.

---

##  Technologies & Tools Used

- **Programming Language:** TypeScript
- **Automation Tool:** Playwright
- **Test Framework:** QAA-OrangeHRM
- **Browser:** Google Chrome
- **Runtime Environment:** Node.js v24.11.1
- **Package Manager:** npm v11.6.2
- **Version Control:** Git & GitHub
- **CI/CD:** GitHub Actions

---

##  Prerequisites

Make sure the following are installed on your machine:

- Node.js v24.11.1
- npm v11.6.2
- Git



##  Installation

```bash
git clone https://github.com/your-username/QAA-ORANGEHRM.git
cd QAA-OrangeHRM
npm install
```



##  Running Tests

```bash
npx playwright test
```



##  Viewing Reports

After execution, open the HTML report:

```bash
npx playwright show-report
```
